Integrates a Horizontal YUI Menu in Wordpress.

== Description ==

This plugin allows the yui menu to be implement into your wordpress templates.

**Features:**

* Currenly implements YUI Version 2.6.0  
* Supports all the args that you can pass into [wp_list_pages()](http://codex.wordpress.org/Template_Tags/wp_list_pages)
* Keyboard and mouse navigation
* CSS Skinning

For more info on YUI see [The Yahoo! User Interface Library (YUI)](http://developer.yahoo.com/yui/)

For more info on YUI Menu see [YUI Menu](http://developer.yahoo.com/yui/menu/)
                   
== Installation ==

1. Upload the folder `wp-yui-menu` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php yui_wp_list_pages('title_li=&sort_column=menu_order');?>` in your templates
